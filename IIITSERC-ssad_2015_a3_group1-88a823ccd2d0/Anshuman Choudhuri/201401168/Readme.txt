* Made Using pygame, Run main.py to start the game

* Consists of two levels, 1st level with one donkey
and 2nd level with Two donkeys

Fireballs swtich between Three speeds randomly

* All necessary functions (stated in comments before the designated function)
have been implemented with different names, getPosition() function is in Person
 Class inherited by classes Donkey and Player


* Consists of 4 modules

* Total of 3 lives in both levels:
	Donkey collision: -3 lives(dead)
	Fire Collision: -1 life

* Once dead the game ends and have to run it again

* After level 2 completion, there is a choice to restart

* Player life is displayed on upper-right corner of the screen
